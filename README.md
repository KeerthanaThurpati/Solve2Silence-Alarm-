Solve2Silence is a Python-based smart alarm system designed to improve user alertness by requiring a mathematical problem to be solved in order to stop the alarm. Unlike traditional alarms that can be easily dismissed, this system ensures active engagement and reduces oversleeping.

🎯 Objective

The objective of this project is to develop an intelligent alarm system that promotes wakefulness by incorporating a problem-solving mechanism. It also demonstrates the implementation of core Python programming concepts in a real-time application.

🚀 Features
⏱️ Real-time alarm functionality
🔢 Random math challenge to stop the alarm
🔊 Continuous alarm sound until correct answer
⚠️ Error handling for invalid inputs
💻 Simple command-line interface.

Technologies Used
Python
Libraries:
time
random
winsound (Windows) / playsound (Cross-platform)

⚙️ How It Works
User sets the alarm time (HH:MM format)
System continuously checks current time
When time matches, alarm starts ringing
User must solve a math problem
Alarm stops only after correct answer
